# Guava project page

Project page for **Guava: Effective and General Harness for Embodied Manipulation**.

Live at: <https://xirui-li.github.io/guava-website/>

## Local preview

```sh
# any static-file server works; here are two options
python3 -m http.server 8000
# or
npx serve .
```

Then open <http://localhost:8000>.

## Structure

```
index.html        — page markup
style.css         — styles (Inter sans, mydarkblue accent from the LaTeX)
images/           — figures (PNGs exported from the paper PDFs)
videos/           — robot rollout videos (H.264, 720p, ~720p, ~1–3 MB each)
```
